# Dhvani Cast - Immersive 3D Website

A modern, immersive 3D website for the Dhvani Cast app, showcasing its features and capabilities in making communication easy and immersive.

## Features

- Immersive 3D scene with Three.js and React Three Fiber
- Interactive animations using GSAP and Framer Motion
- Responsive design for all device sizes
- Modern UI with green and black color theme
- Smooth scrolling and transitions

## Technologies Used

- React
- Vite
- Three.js
- React Three Fiber
- TailwindCSS
- GSAP
- Framer Motion
- Lucide React (for icons)

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/yourusername/dhvani-cast-website.git
   ```

2. Navigate to the project directory
   ```bash
   cd dhvani-cast-website
   ```

3. Install dependencies
   ```bash
   npm install
   ```

4. Start the development server
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173`
